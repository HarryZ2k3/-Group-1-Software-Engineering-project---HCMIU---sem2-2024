Download PostgreSQL with the version: pg from util-linux 2.33.1
Step 1: 
+ Create an account pgAdmin4 with following information: 
+ user: 'postgres'
+ host: 'localhost'
+ database: 'STUDENTRECORDS'
+ port: 5432

Step 2: Click the database 'STUDENTRECORDS' -> click "querry tools" button -> take all data in file "SavingRecords.sql", put them into query part and execute all the code lines.

Step 3: Click the database 'STUDENTRECORDS' -> click "querry tools" button -> take all data in file "INSERT_INTO.sql", put them into query part and execute all the code lines. 

Note: turn on the pgAdmin4 app during the time setting up the app.
